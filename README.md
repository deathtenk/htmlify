## Installation

Add this line to your application's Gemfile:

```ruby
gem 'htmlify'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install htmlify

## Usage

```ruby
  html :div, "Hello World!", id: 'window'
  # returns <div id='window'> "Hello World!"</div>
```
